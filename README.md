# TCR Clustering Demo

This repository contains a demonstration of TCR clustering using `tcrClustR`.

## Viewing the Analysis

The full analysis report is available at this repository's GitHub Pages site:

[**View the Report**](https://GWMcElfresh.github.io/tcrClustRDemo/)

## Running Locally

To run this analysis locally:

1.  Clone this repository.
2.  Open `TCR_Clustering_Demo.Rproj` in RStudio.
3.  Open `index.Rmd`.
4.  Ensure you have the required packages installed (`tcrClustR`, `tidyverse`, `Rdiscvr`, `rmdformats`, `here`, `ComplexHeatmap`, `circlize`, `lme4`, `ggrepel`, `egg`, `scales`, `CellMembrane`).
5.  Knit the document.

The script will automatically download the required data (`data/CD8_Stims.rds`) if you have access to the source.
